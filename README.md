# WarehouseCam — Фото для склада

Мобильное веб-приложение для фотографирования запчастей на складе и автоматической синхронизации ссылок в Google Таблицы.

---

## Что делает система

1. **Кладовщик** открывает приложение на телефоне, вводит артикул, делает фото.
2. Фото загружается в папку `Avito/<артикул>/` на Яндекс.Диске.
3. Файл автоматически публикуется (получает публичную ссылку).
4. **Google Apps Script** (раз в сутки, в 8:00) сканирует папки Яндекс.Диска и заполняет столбец `Ссылка на фото` во всех листах Google Таблицы.

## Установка и запуск

### Требования

- Node.js 20+
- pnpm 9+
- Токен Яндекс.Диска с правами `cloud_api:disk.write`

### Локальный запуск

```bash
# Установить зависимости
pnpm install

# Задать переменную окружения
export YANDEX_TOKEN=y0_ваш_токен

# Запустить бэкенд (порт 8080)
pnpm --filter @workspace/api-server run dev

# Запустить фронтенд (в отдельном терминале)
pnpm --filter @workspace/warehouse-cam run dev
```

### Регенерация API-клиента

```bash
pnpm --filter @workspace/api-spec run codegen
```

## Google Apps Script

### Установка

1. Откройте Google Таблицу с базой запчастей.
2. **Расширения → Apps Script** → вставьте содержимое `attached_assets/yandex_disk_sync.gs`.
3. В меню скрипта: **Запустить → setupTrigger** — создаст ежедневный триггер в 8:00.
4. В **Свойствах скрипта** добавьте:
   - `yandexToken` — OAuth-токен Яндекс.Диска

### Настройка

В начале скрипта (`CONFIG`):

| Параметр | Значение по умолчанию | Описание |
|----------|----------------------|---------|
| `articleColumn` | `2` | Столбец B — артикул |
| `resultColumnHeader` | `"Ссылка на фото"` | Заголовок столбца для ссылок |
| `startRow` | `3` | Строка начала данных |
| `yandexBaseFolder` | `"Avito"` | Корневая папка на Яндекс.Диске |
| `startAfterSheet` | `"Справочник"` | Листы до этого пропускаются |

## Получение токена Яндекс.Диска

1. Перейдите на [yandex.ru/dev/disk/poligon](https://yandex.ru/dev/disk/poligon/)
2. Нажмите **Получить OAuth-токен**
3. Войдите в аккаунт Яндекс
4. Скопируйте токен (начинается с `y0_...`)

Токен имеет полный доступ к Яндекс.Диску (чтение + запись).

## API эндпоинты

### `POST /api/warehouse/photo`

Загрузить фото для артикула.

**Тело запроса (multipart/form-data):**
- `article` (string) — артикул товара
- `photo` (file) — файл изображения

**Ответ:**
```json
{
  "success": true,
  "filename": "photo_1234567890.jpg",
  "publicUrl": "https://yadi.sk/d/..."
}
```

### `GET /api/warehouse/photos/:article`

Получить список фото для артикула.

**Ответ:**
```json
{
  "article": "3ХА501",
  "photos": [
    {
      "name": "photo_1234567890.jpg",
      "publicUrl": "https://yadi.sk/d/...",
      "previewProxyUrl": "/api/warehouse/photo-proxy?path=..."
    }
  ]
}
```

### `GET /api/warehouse/catalog`

Получить все папки каталога Avito.

**Ответ:**
```json
{
  "folders": [
    {
      "article": "3ХА501",
      "photoCount": 6,
      "coverProxyUrl": "/api/warehouse/photo-proxy?path=..."
    }
  ]
}
```

### `GET /api/warehouse/photo-proxy?path=<disk_path>`

Прокси для отображения изображений с Яндекс.Диска в браузере (добавляет OAuth-заголовок).
